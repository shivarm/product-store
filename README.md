## product-store

Highlights:

- 🌟 Tech stack: PERN + TailwindCSS + Daisy UI
- 🚀 Rate Limiting & Bot Detection
- 👌 Global state management with Zustand
- 🐞 Error handling both on the server and client

### Setup .env file

```js
PORT=3000

PGUSER=...
PGPASSWORD=...
PGHOST=...
PGDATABASE=...

ARCJET_KEY=...
ARCJET_ENV=development
```

### Run the API

```bash
npm run dev
```

### Run the frontend

```bash
cd frontend
npm run dev
```
